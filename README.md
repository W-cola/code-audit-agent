# 企业级代码仓库智能审计与合规Agent平台
面向中小软件企业的多Agent协同式代码合规分析工具，解决多仓库管理、安全审计与架构债务治理的痛点。

## 项目背景
解决企业级项目中多仓库代码规范不一致、安全漏洞难排查、合规审计成本高、跨项目依赖关系梳理难的行业痛点。

## 核心架构
- **代码扫描Agent集群**：批量拉取企业项目仓库，进行全量代码的语法解析、依赖分析与安全漏洞扫描
- **长链推理分析Agent**：跨文件、跨仓库长上下文关联推理，识别代码异味、合规风险点与架构债务
- **报告生成与修复Agent**：自动生成合规审计报告、漏洞修复建议与代码重构PR

## 快速开始
1.  安装依赖：`pip install -r requirements.txt`
2.  配置环境变量（可选）：复制 `.env.example` 为 `.env` 并填入API Key
3.  运行项目：`python main.py`
